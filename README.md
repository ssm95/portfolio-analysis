# Portfolio Performance & Risk Analysis

## Overview
This project analyses the historical performance and risk of a diversified investment portfolio using Python.

## Objective
To evaluate portfolio performance using:
- Historical returns
- Volatility
- Correlation between assets
- Basic risk metrics

## Assets Included
- S&P 500 ETF
- FTSE 100 ETF
- Apple (AAPL)
- Microsoft (MSFT)
- Government Bond ETF

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Status
Project in progress.
